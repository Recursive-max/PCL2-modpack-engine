# PCL2 整合包推荐引擎

⚔ PCL2 启动器的 Minecraft 整合包智能推荐主页，Gen X Soft Club 暗黑中世纪风格。

## 🔗 PCL2 使用方式

在 PCL2 启动器 → 设置 → 个性化 → 自定义主页 中填入：

```
https://2401-84211770.gitcode.io/PCL2-modpack-engine/Custom.xaml
```

## 📊 数据来源

- B站 — 整合包视频与UP主数据
- BBSMC — 中文MC资源下载站
- CurseForge — 全球最大MC模组平台
- Modrinth — 开源MC整合包平台

## ⚙ 技术栈

- XAML (PCL2 原生)
