# 安装与使用教程

> PCL 整合包推荐引擎 — 如何在 PCL2 中配置自定义主页

---

## 联网使用（推荐）

1. 打开 **Plain Craft Launcher 2**
2. 点击右上角 **设置** ⚙️
3. 进入 **个性化 → 自定义主页**
4. 在 **「联网更新」** 区域，点击 **「下载地址」**
5. 输入以下地址：

```
https://raw.gitcode.com/2401_84211770/PCL-modpack-engine/raw/main/output/Custom.xaml
```

6. 点击 **「确定」**，等待下载完成
7. 返回启动页，主页即加载

---

## 本地使用

1. 在 [GitCode 仓库](https://gitcode.com/2401_84211770/PCL-modpack-engine) 下载 `output/Custom.xaml`
2. 将文件放入 PCL2 目录下的 `Custom` 文件夹（没有则新建）
3. 启动 PCL2，主页即从本地加载

---

## 刷新主页

- **PCL2 内置**：点击主页上的 **「刷新」** 按钮
- **手动刷新**：设置 → 个性化 → 自定义主页 → 联网更新 → 重新下载
- **强制刷新**：按 `F5` 或重启 PCL2

---

## 常见问题

### Q：加载后显示空白 / 加载失败

**原因**：
- 网络不可达（GitCode Raw 被墙）
- 下载未完成
- PCL2 版本过低（需要 v2.5.0+）

**排查步骤**：
1. 检查网络连接是否能访问 `gitcode.com`
2. 检查浏览器是否能打开下载地址
3. 确认 PCL2 版本 ≥ 2.5.0
4. 尝试切换网络（手机热点 / 代理）
5. 切换到「本地使用」方式

### Q：主页内容不是最新的

主页每日 07:00 (UTC+8) 自动更新。手动触发更新：
- 设置 → 个性化 → 自定义主页 → 重新下载

### Q：某个整合包链接失效了

请通过 [反馈入口](https://gitcode.com/2401_84211770/PCL-modpack-engine/issues) 提交 Issue，我们会尽快修复。
